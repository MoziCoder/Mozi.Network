# Mozi.Network

Mozi.Network是基于.Net开发的网络应用组件套装。

## 功能集合
- [HttpEmbedded][httpembedded] Http服务器
- [SSDP][ssdp] SSDP/UPNP实现
- [StateService][stateservice] 心跳服务组件
- [Telnet][telnet] Telnet服务器及客户端实现


## 项目地址

- [Github][github]

- [Gitee][gitee]

- [CSDN][codechina]

## 程序下载

~~~shell

	dotnet add package Mozi.HttpEmbedded --version 1.2.5

~~~
